Game Controls Guide:
--------------------
| Action               | Control                         |
|----------------------|---------------------------------|
| Move Character       | W / A / S / D                   |
| Jump                 | Spacebar                        |
| Light Attack         | Left Mouse Button               |
| Heavy Attack         | Right Mouse Button              |
| Interact (Shrines)   | I                               |
| Dodge / Roll         | E                               |
| Delete Save Slot     | X                               |
| Sprint               | Shift + Movement Keys (WASD)    |
